# MemHoard Web App

## Overview
**MemHoard** is a secure, on-premise AI memory layer designed for enterprise environments. This web application serves as the frontend interface, featuring a simulated "Neural OS" experience.

## Features
- **Sovereign AI:** Full offline capabilities and data control.
- **Neural Interface:** A simulated OS environment with modules for Stream, Search, and Knowledge Graph.
- **Visuals:** High-end shader backgrounds (ColorBends) and smooth animations.
- **Tech Stack:** React, TypeScript, Vite, Tailwind CSS v4, Three.js.

## Getting Started

### Prerequisites
- Node.js (v20+)
- npm

### Installation
```bash
git clone https://github.com/cmpdpucc/uderly-memhoard-web.git
cd uderly-memhoard-web
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Styling
This project uses **Tailwind CSS v4**.
- Configuration is handled in `src/index.css` via the `@theme` directive.
- Global styles are in `src/styles/global.scss`.
